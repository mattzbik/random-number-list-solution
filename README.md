# random-number-list-solution

## Run Locally

Clone the project

```bash
  git clone https://github.com/mattzbik/random-number-list-solution.git
```

Go to the project directory

```bash
  cd random-number-list-solution
```

Install dependencies

```bash
  npm install
```

Start the program

```bash
  npm run dev
```

Test the program

```bash
  npm run test
```
